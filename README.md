# trb-core

## Запуск

### С помощью Docker

### Используемые порты

- 8080 - сервис
- 5430 - база данных

```bash
docker-compose up --build
```

### API

- API в формате 3.0.1 доступно в [файле](./docs/api-docs.yaml)
- Swagger после запуска доступен по [адресу](http://localhost:8080/swagger-ui/index.html)
